# `.claude` — 에이전트·Claude Code 참고

이 폴더는 **에이전트가 새 작업을 할 때 읽을 보조 문서**를 둔다.

| 파일 | 내용 |
|------|------|
| **`QUANT_STRATEGIES.md`** | Quant 전략 추가 시 폴더 구조, 체크리스트, tradingview 와의 데이터 차이 |

프로젝트 전체 스택·아키텍처는 저장소 루트의 **`../CLAUDE.md`** 를 본다.

> 참고: `tradingview_mcp` 의 `.claude/` 아래에는 설정 파일만 있고, 실질적인 가이드는 **`tradingview_mcp/CLAUDE.md`** (루트)에 있었다.  
> 이 레포는 루트 `CLAUDE.md` + 이 폴더의 `QUANT_STRATEGIES.md` 로 역할을 나눔.
